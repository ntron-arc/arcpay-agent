## 🎯 What is ArcPay Agent?

ArcPay Agent is a fully autonomous payment agent built on Arc L1 blockchain. It uses programmable logic to research, decide, and execute USDC transactions on behalf of users — enabling a new class of agentic economic experiences.

The agent manages its own Circle wallet, maintains a USDC budget, and autonomously approves or rejects payments based on built-in decision logic — all settled on Arc Testnet with real-time transaction tracking.

## 🛠️ Circle Products Used

| Product | Usage |
|---|---|
| **USDC** | Settlement currency for all agent payments |
| **Circle Wallets** | Agent wallet creation and key management |
| **Arc Testnet** | L1 blockchain for transaction execution |

## ✨ Key Features

- 🤖 Autonomous Decision Engine — Agent approves/rejects based on amount, reason, and budget
- 💸 Real-time USDC Payments — Instant settlement on Arc Testnet
- 🧾 Transaction History — Full activity log
- 📊 Live Budget Tracking — Agent monitors remaining USDC balance
- 🔒 Safety Limits — Auto-reject transactions above 50 USDC
